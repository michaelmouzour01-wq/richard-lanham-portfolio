# Portfolio Design Brainstorm - Richard Lanham, PhD

## Design Approach Selected: Modern Technical Elegance

**Design Movement:** Contemporary Technical Minimalism with Scientific Precision

**Core Principles:**
1. **Data-Driven Aesthetics** – Visual hierarchy mirrors information hierarchy; typography and spacing reflect scientific rigor
2. **Restrained Color Palette** – Deep navy, charcoal, and accents of teal/cyan evoke precision instruments and technical depth
3. **Asymmetric Layouts** – Alternating content blocks and strategic whitespace create visual rhythm without rigidity
4. **Functional Typography** – Serif headings convey authority; sans-serif body text ensures readability

**Color Philosophy:**
- **Primary Background:** Deep navy (#0F1419) – suggests depth, professionalism, and technical sophistication
- **Accent Color:** Teal/Cyan (#00D9FF) – represents innovation, data visualization, and energy
- **Text:** Off-white (#F5F5F7) on dark backgrounds for contrast; dark charcoal on light sections
- **Emotional Intent:** Trustworthy, precise, forward-thinking; conveys expertise without arrogance

**Layout Paradigm:**
- Hero section with asymmetric split: large typography on left, embedded video/visualization on right
- Alternating content blocks: full-width sections with staggered left/right text and imagery
- Card-based project showcase with hover animations revealing technical details
- Timeline or vertical progression for experience and education

**Signature Elements:**
1. **Geometric Accent Lines** – Subtle diagonal lines and borders in teal connecting sections
2. **Data Visualization Badges** – Small charts/graphs representing technical skills (Python, MATLAB, etc.)
3. **Code Snippet Callouts** – Highlighted technical achievements in monospace font

**Interaction Philosophy:**
- Smooth scroll-triggered animations that reveal content progressively
- Hover states on project cards that expand to show technical details
- Subtle parallax effects on hero section to create depth
- Smooth transitions between sections with fade-in effects

**Animation:**
- Fade-in on scroll for content blocks (staggered timing)
- Smooth color transitions on hover (0.3s ease)
- Subtle scale transforms on interactive elements (1.02x on hover)
- Progress indicators that animate as user scrolls through sections

**Typography System:**
- **Display Font:** Playfair Display (serif) for main headings – conveys authority and precision
- **Body Font:** Inter (sans-serif) for all body text – clean, professional, highly readable
- **Accent Font:** IBM Plex Mono for code snippets and technical callouts
- **Hierarchy:** H1 (3.5rem), H2 (2.5rem), H3 (1.75rem), Body (1rem)
